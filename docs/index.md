# bankstatementparser-loader-mt942

SWIFT MT942 _Interim Transaction Report_ loader for the
[bankstatementparser](https://github.com/sebastienrousseau/bankstatementparser)
library. Parses MT942 text into `bankstatementparser` `Transaction`
objects — a format the core library does not support.
